AI线上课程系统 - 使用说明
================================

一、快速开始
------------

1. 安装依赖包
   打开命令行，进入项目目录，运行：
   pip install -r requirements.txt

2. 配置API密钥
   编辑 ai_course_system/config.py 文件
   将 DEEPSEEK_API_KEY 设置为您的API密钥

3. 运行程序
   方式1: 直接运行Python脚本
   cd ai_course_system
   python main.py

   方式2: 打包为EXE后运行
   python build.py
   运行 dist/AI课程系统.exe


二、功能说明
------------

【学生功能】
1. 我的课程
   - 浏览所有可选课程
   - 选择感兴趣的课程
   - 查看课程资料

2. AI问答
   - 选择已选课程
   - 输入问题，AI基于课程资料回答
   - 查看历史问答记录

3. 在线测验
   - 参加教师发布的测验
   - 自动批改客观题
   - AI批改主观题并给出反馈

4. 学习记录
   - 查看选课情况
   - 查看提问历史
   - 查看测验成绩

【教师功能】
1. 课程管理
   - 创建新课程
   - 上传课程资料（PDF/Word/TXT）
   - 管理课程文档
   - 删除课程

2. AI问答
   - 使用AI助教
   - 查看学生提问记录
   - 了解学生学习难点

3. 试卷管理
   - 输入知识点，AI自动生成试卷
   - 查看测验统计数据
   - 分析学生成绩分布

4. 数据统计
   - 查看课程统计
   - 查看学生人数
   - 查看提问和测验数据


三、注意事项
------------

1. API密钥配置
   - 必须配置有效的DeepSeek API密钥
   - 密钥在 ai_course_system/config.py 中配置
   - 没有密钥将无法使用AI功能

2. 文档上传
   - 支持格式: PDF, DOCX, TXT
   - 建议单个文件不超过50MB
   - 文档内容会被提取用于AI问答

3. 数据存储
   - 所有数据存储在 data 目录下
   - 定期备份 data/course_system.db 文件
   - 删除数据库文件将清空所有数据

4. 网络要求
   - AI功能需要网络连接
   - 确保能访问DeepSeek API服务
   - 网络不稳定可能导致AI响应慢


四、打包说明
------------

方法1: 使用打包脚本（推荐）
   python build.py
   
   打包完成后：
   - 可执行文件: dist/AI课程系统.exe
   - 发布包: release/ 目录

方法2: 手动打包
   pyinstaller build_simple.spec
   
   或使用完整命令：
   pyinstaller --name="AI课程系统" --windowed --onefile --add-data "ai_course_system;ai_course_system" ai_course_system/main.py


五、常见问题
------------

Q1: 程序无法启动？
A: 检查是否安装了所有依赖包，运行 pip install -r requirements.txt

Q2: AI不回答问题？
A: 检查API密钥是否正确配置，网络是否连接正常

Q3: 无法上传文档？
A: 检查文件格式是否支持，文件是否损坏

Q4: 忘记密码？
A: 目前需要删除数据库重新注册，或联系管理员

Q5: 打包后程序无法运行？
A: 确保使用了正确的打包命令，包含了所有必要的文件


六、系统要求
------------

- 操作系统: Windows 10/11
- Python版本: 3.8 或更高
- 内存: 至少 2GB RAM
- 磁盘空间: 至少 500MB
- 网络: 需要互联网连接（AI功能）
